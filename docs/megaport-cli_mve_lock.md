# lock

Lock an MVE

## Description

Lock an MVE to prevent modifications.

This command locks a Megaport Virtual Edge (MVE) to prevent any changes from being made to it. Use the unlock command to re-enable modifications.

### Example Usage

```sh
  megaport-cli mve lock [mveUID]
```

## Usage

```sh
megaport-cli mve lock [flags]
```

# unlock

Unlock an MCR

## Description

Unlock a previously locked MCR.

This command unlocks a Megaport Cloud Router (MCR) that was previously locked, allowing modifications to be made again.

### Example Usage

```sh
  megaport-cli mcr unlock [mcrUID]
```

## Usage

```sh
megaport-cli mcr unlock [flags]
```

# unlock

Unlock an MVE

## Description

Unlock a previously locked MVE.

This command unlocks a Megaport Virtual Edge (MVE) that was previously locked, allowing modifications to be made again.

### Example Usage

```sh
  megaport-cli mve unlock [mveUID]
```

## Usage

```sh
megaport-cli mve unlock [flags]
```

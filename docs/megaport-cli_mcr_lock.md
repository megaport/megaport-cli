# lock

Lock an MCR

## Description

Lock an MCR to prevent modifications.

This command locks a Megaport Cloud Router (MCR) to prevent any changes from being made to it. Use the unlock command to re-enable modifications.

### Example Usage

```sh
  megaport-cli mcr lock [mcrUID]
```

## Usage

```sh
megaport-cli mcr lock [flags]
```

# search

Search locations by name (fuzzy match)

## Description

Search for locations by name using fuzzy matching.

This command searches for locations whose names match the provided search term. The search uses fuzzy matching, so partial or approximate names will return results.

### Example Usage

```sh
  megaport-cli locations search "Equinix"
  megaport-cli locations search "Sydney"
```

## Usage

```sh
megaport-cli locations search [flags]
```

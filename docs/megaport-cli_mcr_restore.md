# restore

Restore a deleted MCR

## Description

Restore a previously deleted MCR.

This command allows you to restore a previously deleted MCR, provided it has not yet been fully decommissioned.

### Example Usage

```
  restore [mcrUID]
```


## Usage

```
megaport-cli mcr restore [flags]
```

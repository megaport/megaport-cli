# get

Get details for a specific user

## Description

Get details for a specific user by employee ID.

This command retrieves and displays detailed information about a specific user.

### Example Usage

```sh
  megaport-cli users get 12345
```

## Usage

```sh
megaport-cli users get [flags]
```

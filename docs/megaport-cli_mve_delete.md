# delete

Delete an existing MVE

## Description

Delete an existing Megaport Virtual Edge (MVE).

This command allows you to delete an existing MVE by providing its UID.

### Example Usage

```
  delete [mveUID]
```


## Usage

```
megaport-cli mve delete [mveUID] [flags]
```

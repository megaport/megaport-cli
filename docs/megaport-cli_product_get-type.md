# get-type

Get the type of a product by UID

## Description

Get the type of a product based on a product UID.

This command retrieves and displays the product type for the specified product UID.

### Example Usage

```sh
  megaport-cli product get-type [productUID]
```

## Usage

```sh
megaport-cli product get-type [flags]
```

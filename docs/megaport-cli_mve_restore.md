# restore

Restore a deleted MVE

## Description

Restore a previously deleted MVE.

This command allows you to restore a previously deleted Megaport Virtual Edge (MVE), provided it has not yet been fully decommissioned.

### Example Usage

```sh
  megaport-cli mve restore [mveUID]
```

## Usage

```sh
megaport-cli mve restore [flags]
```

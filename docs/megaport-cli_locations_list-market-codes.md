# list-market-codes

List all market codes

## Description

List all market codes used to categorize Megaport locations.

This command fetches and displays a list of all market codes available in the Megaport API.

### Example Usage

```sh
  megaport-cli locations list-market-codes
```

## Usage

```sh
megaport-cli locations list-market-codes [flags]
```

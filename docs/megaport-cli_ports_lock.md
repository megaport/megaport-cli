# lock

Lock a port

## Description

Lock a port in the Megaport API.

This command allows you to lock an existing port by providing the UID of the port as an argument.

Example usage:

```
  megaport-cli ports lock [portUID]

```



## Usage

```
megaport-cli ports lock [portUID] [flags]
```
